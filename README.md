# Code Snippet Generator

## About

As programmers, we are trained to hate repetitive tasks. But still, we have to do it way too much. Use the code snippets generator to generate repetitive but useful/mandatory code with a GUI for your application.

#### Live Site

[https://code-snippet-generator1.netlify.app/](https://code-snippet-generator1.netlify.app/)

## Installation

```bash
npm i
```
